Info (UT)

UT :: Actor (UT) >> Info (Package: Engine)

This is the abstract base class for... uhh... everything that doesn't fit in anywhere else. (Since by principle all classes "hold information," there's not much more to say for it.)

The Info class doesn't declare new or override a parent class's properties or methods.

See Info for the UT2003 version of this class.
