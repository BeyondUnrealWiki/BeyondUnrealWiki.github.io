Mod Ideas/CorpseRedemption

Mod ideas for UT / Fun little "redeem your life" mod. - By Olga

Description

My idea is for some kind of modification that makes it so that after you die, you lose a frag and your corpse starts running around. If you frag your corpse, you redeem a frag. I don't know what to make this as, I'm thinking either a mut or a gametype though. Dunno what UT to make this for though.

UPDATE: I think I might actually start coding this old idea, which I forgot completely about.

- Olga

Interested Scripters

If you are interested in developing this mod for UT then add your name to the list. Once you start development you should indicate that below (and hopefully include a link to a journal page). Before you start development you should also check this section to see if anyone else has started.
