Mod Ideas/Trampoline

Mod ideas for UT 2003 – Trampoline – the new jump pad

Description

The Trampoline behaves in a similar way to a jump pad with the following differences. If you press jump while on the trampoline you will increase the height you bounce. If you press crouch while on the trampoline you will decrease the height you bounce.

Interested Scripters

If you are interested in developing this mod for UT2003 then add your name to the list. Once you start development you should indicate that below (and hopefully include a link to a journal page). Before you start development you should also check this section to see if anyone else has started.

Discussion

ArcaneSpeech: Seems to me that timing your jumps would be more realistic, although vulnerable to laggy situations...

ZxAnPhOrIaN: This should not bounce you if you walk on to it, like in real life. It should make the player wobble back and forth when not bouncing.

Legal: Couldn't this be done with a trigger and a kicker?

EntropicLqd: Using the double jump feature in UT2003 you could probably implement this with a custom Jump Pad. All you'd need is a jump pad that provided a Z boost equal do the player's downard Z movement at the point of collision. Players could then use a double jump to gain height. The only downside of that is that it would take a while to gain any height. Maybe a better way would be to automatically bounce the player higher - and forget about having the player need to do anything.

Legal: If so, it'd be easier to have the pad take your ierted Z-velocity*1.4 and done. Easier, no?

ZxAnPhOrIaN: Should there be a dynamic water plane or something similar (karma?) as the thing bounces?

Foxpaw: I don't think that either of those would really give the desired effect.. I think a scripted skeletal animation would be the best choice for the visuals if you want it to be entirely realistic. Regular vertex animation would probrably be fine anyway, I don't think it would be imperative that the visuals be 100% consistant with the players motion.
