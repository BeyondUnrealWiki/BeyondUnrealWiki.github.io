SpecialMover

UT2003 :: Actor >> Brush >> Mover >> SpecialMover (custom)

The SpecialMover allows you to kill bots without triggering death-zones. This fixes a problem which existed in the original UT from epic.

Download

[SpecialMover_v05.zip] (2,8 kB)

Usage

NOTE: If you want to use the SpecialMover in UnrealED then you have to edit your UnrealTournament.ini! Add a line EditPackages=SpecialMover (where the other ones are).

You can use the SpecialMover like a normal mover. Just create the brush for your mover as you would do it for a normal mover, but add the SpecialMover instead (right click on add mover icon).
