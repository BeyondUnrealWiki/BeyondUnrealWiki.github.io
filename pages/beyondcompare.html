BeyondCompare

A rather advanced file/folder comparison tool.

Among the things it can do:

  • Define "filters" to decide what's a true changed line, and changes to ignore
  • Side by side file comparison with color-coding
  • Comparison view lets you move lines between files, insert them, edit them, etc.

Available at http://www.scootersoftware.com/

Xhiris: I've been using this to compare the Deus Ex classes with the UT ones, and it's been a lifesaver. It's got a "diff" view a lot like [Visual SourceSafe]?, but it even shows changes at the character level. I'm actually considering this for work, it's that good.
