Mapping For One Flag Capture

Is a little like Mapping For CTF exept for the objectives.

Actor >> NavigationPoint >> JumpDest >> JumpSpot >> GameObjective >> OFC_Goals?
  • OFC_RedGoal replace the Red flag.
  • OFC_BlueGoal replace the Blue flag.

The goals are the capture point of flag, one alone for each team.

Actor >> NavigationPoint >> JumpDest >> JumpSpot >> GameObjective
  • OFC_FlagBase place one or several in middle map, the flag start and restart (after a capture) randomly on one Flag base.

For all details of all specific actor of this mod see : http://oneflagcapture.free.fr/mapping.htm
