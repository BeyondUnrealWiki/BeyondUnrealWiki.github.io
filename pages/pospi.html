Pospi

About me

You can find my website at [pospi.spadgos.com]. I consider my main focus to be high-level (mostly gameplay) programming, though I do tend to branch out into other areas like game design, 2D & 3D artwork and level design. I've been an Unreal fan since the very beginning and have been creating content since some time in 2003.

At the moment I work as a PHP web developer which is fairly balls but it pays the bills.

I haven't really contributed anything to the wiki at this point, but it's about time I started giving something back to the fantastic community that fostered my wonderful experiences in game development.

While working on my last project, I made a pact with myself that it would be the last mod I made for UT2004, so I'm being fairly inactive at the moment and will resurface more when we all get a new game and a new engine to play with.

UT2004 mods

Large undertakings (mods and TCs)

Furious Steals
http://furious.spadgos.com/
A driving slash beat-em-up TC
DM-MCE-Relativity
http://pospi.spadgos.com/index.php?pg=games&sec=10
A map and toolset that allows for 'proper' control over gravity direction. Think 4-dimensional gameplay.
Marshmallow Duel - Mowbray's Revenge
http://mduel2k5.spadgos.com/
Supposed to be a remake of the 1996 game [Marshmallow Duel] but the transition to 3D is not quite successful IMO. I created a more faithful sequel for the [Nintendo DS].
Soldat UT
http://mduel2k5.spadgos.com/files.php
Sidescrolling combat for UT2004. An offshoot of Mowbray's Revenge since it was mainly a test package for it.
EscherSketch
http://pospi.spadgos.com/projects/grabin
A test implementation of gestural input. On hold, but you can download a very alpha version.
Somos
http://pospi.spadgos.com/index.php?pg=games&sec=4
Just a player model. My first project for UT2Kx.
