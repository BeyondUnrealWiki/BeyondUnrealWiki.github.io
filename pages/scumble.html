After quite a long period, I've picked up on UnrealScript again. Managed to get further into the code this time and tried out some subclassing. During a few hours, with the help of WOTgreal, I built a package with a test gametype in order to use my own Bot subclass. My initial motivation was to customise the debugging displays to help me understand the Bot AI, although I was led into other areas. I was fairly pleased with the fact that I had few errors, and my test DeathMatch worked as expected.
