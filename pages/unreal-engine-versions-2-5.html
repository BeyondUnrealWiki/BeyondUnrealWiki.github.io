Unreal Engine Versions/2.5

Delete Me - Refactored into Unreal Engine 2

Unreal Engine 2.5 targets 2004~2005 high-end PCs. an enhanced version of Unreal Engine 2 with an optimized rendering engine. core technology re-written and improving and extension, The Unreal Engine 2.5 adds support for 64-bit Windows and 64-bit Linux operating systems. Improving and enhancing UnrealEd toolsets. One fork of the Engine is also highly optimized for the Xbox hardware shader pipeline based on a few graphical enhancements, The Xbox memory management system, Xbox GUI system, editor, and Xbox live support. These Xbox optimized features are integrated into the Unreal Engine 2X, but is an off-shoot of the Unreal Engine 2.5.

Engine Details

  • Included all of the Unreal Engine 2.0 features

Rendering Technologies

  • Direct3D 9 and same level of OpenGL support
  • While 2.5 does not make extensive use of Direct3D 9 and OpenGL 2 level of features, it does use the Direct3D 9 API and thus provides an easy starting point for licensees interested in adding Direct3D 9 and OpenGL 2 level of graphical features
  • UDN site Unreal Engine 2.5 support also features multipass bump-mapping (normal map/bump map/specular map/diffuse map/gloss map/environment map/opacity map/mask map etc), per-pixel lighting and per-pixel shading, virtual displacement mapping, high dynamic range rendering, soft shadows and other modern graphical features present in the Direct3D 9 and OpenGL 2 feature set
  • 4096x4096 size texture support
  • Improvements rendering engine speed
  • Enhanced BSP/CSG and PVS/Portal world rendering technology and Scene graph and all Scene Management
  • Software rendering with Pixomatic software renderer (no additional cost)

Other Features

  • Included particle system editor
  • Voice-over IP support
  • Text-to-speech support
  • AI speech recognition
  • Vehicles optimized physics
  • Network code re-written
  • Enhancing physics performance
  • OC3 Entertainment's Impersonater (no additional cost)
  • Softimage's XSI Mod Tool Plug-in
  • Included IDV's SpeedTreeRT (no additional cost)
  • UDN site support on other toolsets, utilities and sample contents
