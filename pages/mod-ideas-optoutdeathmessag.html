Mod Ideas/OptOutDeathMessages

Mod ideas for Any :: UT :: :: UT2003 :: – OptOutDeathMessages

Description

This mutator will eliminate all DeathMessages that show up in the Chat area. ie. "SomePlayer electrified OtherPlayer with SomeElectricThing" Who needs to see that? Well, this would make it possible for server admins to eliminate them. Maybe this could run client-side, but I don't know about anti-cheat software allowing it.
