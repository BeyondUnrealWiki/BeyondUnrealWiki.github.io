Console Bar

The console bar is the area at the bottom of the UnrealEd Interface.

UnrealEd 3

UnreaEd 3 console bar (squished to fit)
  1. UnrealEd's console. Text commands can be typed in here. Most of them are equivalent to actions in the rest of the interface, but some of UnrealEd's features can only be accessed here. See UnrealEd Console for the list.
  2. Opens the log window. Some console commands produce output here.
  3. funny lock icon
  4. Snap to vertex toggle. Green when active.
  5. Snap to grid toggle. Green when active.
  6. Grid size. See Snap to Grid or Grid for more on using the grid.
  7. Snap to Rotation Grid toggle.
  8. Maximize active viewport toggle.
  9. DrawScale3D – these 3 boxes show the value of the Display → DrawScale3D property for the currently selected Actor(s). Useful for quickly changing the size of Static Meshes.
