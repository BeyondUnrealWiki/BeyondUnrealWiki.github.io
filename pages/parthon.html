About me

I'm a long time lurker for the wiki and I thought I may as well finally introduce myself.

I only came into the UT scene with UT2k4 unfortunately. Missed out on all the good UT fun back when that came out. Everytime a new game comes out I try and see how to mod it and I've realised that UT2k4 is a great game with an awesome code base. I come from coding with Quake 3, which is just so damn unflexable when it comes to large modifications, and from making my own games, which are way too much effort.

I look forward to posting heaps of information to the wiki when I learn it.

Oh .. and I live in Perth, Australia.

Mod Info

I've almost finished my first working mod for UT2k4 for those who play it. It's very similar to the Mod Ideas/UnrealBallPool with slightly modified rules.

I call it "MultiBall" but I really need a better name for it when I finish it.

It's like bombing run except there can be multiple ball spawn points. Balls spawn at regular intervals up to a maximum number and the idea is to run them back to your base and capture a certain number of balls at one time to win. And if you are daring enough you can raid the enemy base to steal their balls, if you can make it back to your own base. The gameplay right now is hectic and frantic, but probably only because my test map is so small and that's the way I like it.

Right now it's working well, just needs a little tweaking.

I will have a link when I manage to get the bugs ironed out.
